# ft-python-scripts

## Docker
* To start Chrome container :
```
docker-compose up -d
```

## Python
* To start a script :
```
python python_org_search.py
```

* Add `pip` on mac (Python module installer)
```
sudo easy_install pip
```

* To add a missing module in python
```
sudo python -m pip install selenium
```
